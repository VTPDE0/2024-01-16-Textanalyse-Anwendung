Aufgabenstellung: Textanalyse

Schreibe ein Java-Programm, das einen Text vom Benutzer einliest und verschiedene statistische Informationen über den Text ausgibt. Das Programm sollte die folgenden Funktionen bieten:

- Zeichenzählung: Das Programm sollte die Anzahl der Zeichen im Text zählen, einschließlich Leerzeichen und Satzzeichen.
- Wortzählung: Es sollte die Anzahl der Wörter im Text zählen. Wörter sind durch Leerzeichen getrennt.
- Durchschnittliche Wortlänge: Das Programm sollte die durchschnittliche Länge der Wörter im Text berechnen (Summe der Längen aller Wörter / Anzahl der Wörter).
- Häufigstes Wort: Das Programm sollte das am häufigsten vorkommende Wort im Text identifizieren und seine Anzahl im Text anzeigen.
- Beenden des Programms: Der Benutzer sollte das Programm beenden können.
